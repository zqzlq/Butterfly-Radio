# Butterfly Radio 启动指南

## 环境要求

| 依赖 | 版本 | 说明 |
|------|------|------|
| Node.js | >= 18 | 前端和 Electron 运行时 |
| pnpm | >= 8 | 前端包管理器 |
| Python | 3.11 ~ 3.12 | 后端运行时（3.14 不兼容部分依赖） |
| uv | >= 0.11 | Python 包管理器（推荐） |
| FFmpeg | 任意 | 音频处理（需在 PATH 中或放在 `resources/ffmpeg/`） |

## 首次安装

### 1. 安装前端依赖

```bash
cd frontend
pnpm install
```

### 2. 安装 Electron 依赖

```bash
cd electron-main
npm install
```

### 3. 安装 Python 后端依赖

```bash
cd python-backend

# 使用 uv（推荐）
uv venv --python 3.12 .venv
uv pip install -r requirements.txt

# 或使用 pip
python -m venv .venv
.venv\Scripts\activate    # Windows
pip install -r requirements.txt
```

### 4. 配置 AI

在 `python-backend/` 下创建 `.env` 文件：

```env
# DeepSeek API Key（在 platform.deepseek.com 获取）
DEEPSEEK_API_KEY=sk-你的key
LLM_MODEL=deepseek-v4-flash

# TTS 方案：edge（免费）| volcengine | chat_tts
TTS_PROVIDER=edge
TTS_VOICE=xiaoxiao
```

## 启动方式

### 方式一：分别启动（开发调试）

**终端 1 — Python 后端：**

```bash
cd python-backend
.venv\Scripts\activate
set BUTTERFLY_DEV=1
python main.py
```

后端启动在 `http://127.0.0.1:3000`

**终端 2 — 前端开发服务器：**

```bash
cd frontend
pnpm dev
```

前端启动在 `http://localhost:5173`，浏览器打开即可使用。

**终端 3 — Electron 桌面端（可选）：**

```bash
cd electron-main
npm run dev
```

### 方式二：一键启动

```bash
# 在项目根目录
npm install          # 安装 concurrently
npm run dev          # 同时启动后端 + 前端 + Electron
```

## 验证服务

```bash
# 检查后端
curl http://127.0.0.1:3000/health

# 检查 AI 状态
curl http://127.0.0.1:3000/api/ai/host

# 检查 TTS 音色
curl http://127.0.0.1:3000/api/ai/tts-voices

# 测试 AI 口播（流式）
curl -X POST http://127.0.0.1:3000/api/ai/commentary \
  -H "Content-Type: application/json" \
  -d '{"context":"greeting","stream":true}'
```

## 常见问题

### 端口 3000 被占用

```bash
# 查看占用进程
netstat -ano | findstr :3000

# 终止进程
taskkill /F /PID <进程ID>
```

### Python 3.14 不兼容

部分依赖（如 aiosqlite）不支持 Python 3.14。使用 uv 安装 3.12：

```bash
uv venv --python 3.12 .venv
```

### FFmpeg 未找到

- 下载 FFmpeg 并加入系统 PATH
- 或将 `ffmpeg.exe` 放入 `resources/ffmpeg/` 目录

### Edge TTS 无网络

Edge TTS 需要网络连接（调用微软在线服务）。如需离线使用，切换到 ChatTTS 本地方案。

## 项目结构

```
MyMusic/
├── frontend/           # React + TypeScript 前端
├── python-backend/     # FastAPI 后端
│   ├── ai/             # LLM + TTS 引擎
│   ├── api/            # REST API 路由
│   ├── core/           # 广播调度、Socket.IO、音频引擎
│   ├── db/             # SQLAlchemy 数据库
│   └── main.py         # 后端入口
├── electron-main/      # Electron 桌面端
└── resources/          # 静态资源（音乐、模型、图标）
```
