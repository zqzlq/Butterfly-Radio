# Butterfly Radio — 前端 UI/UX 设计提示词文档

**用途**：交给 Stitch（或同类 AI 设计工具）生成前端界面设计稿
**设计风格**：暗黑风赛博朋克 / Cyberpunk Dark
**分辨率基准**：1440 × 900（桌面端，支持 1280~2560 宽度自适应）
**目标平台**：Electron 桌面应用（Windows / macOS）

---

## 一、全局设计语言

### 1.1 色彩体系

```
主背景色：      #0A0A0F（纯黑微蓝，深空底色）
卡片/面板背景：  #12121A（深灰蓝，轻微磨砂质感）
次级背景：      #1A1A28（悬浮卡片、弹窗背景）
主强调色：      #00F0FF（霓虹青，用于主按钮、ON AIR 标识、频谱高亮）
次强调色：      #FF006E（霓虹粉，用于收藏、心跳、互动高亮）
辅助色：        #7B61FF（霓虹紫，用于标签、次要操作）
文本主色：      #E8E8F0（近白色，正文内容）
文本次色：      #8888A0（灰紫色，时间戳、辅助信息）
禁用态：        #3A3A50（深灰，不可交互元素）
渐变参考：      线性渐变 135deg, #00F0FF → #7B61FF（用于按钮悬停、频谱光效）
```

### 1.2 字体规范

```
标题字体：  Inter / Noto Sans SC Bold, 20~32px
正文字体：  Inter / Noto Sans SC Regular, 14~16px
数字/时间： JetBrains Mono / Source Code Pro, 等宽字体, 14~48px
标签/小字： Inter Medium, 10~12px, 全大写 + 字间距 0.1em
```

### 1.3 圆角与阴影

```
卡片圆角：    16px
按钮圆角：    24px（胶囊型）/ 8px（方型）
输入框圆角：  12px
阴影：        0 4px 24px rgba(0, 240, 255, 0.08)（微弱霓虹辉光）
悬停阴影：    0 8px 32px rgba(0, 240, 255, 0.15)
```

### 1.4 通用视觉效果

- **玻璃拟态**：面板/卡片使用 `backdrop-filter: blur(20px)` + 半透明背景
- **霓虹辉光**：关键元素（ON AIR、频谱、强调按钮）带微弱外发光 `box-shadow: 0 0 12px rgba(0,240,255,0.3)`
- **线条分隔**：使用 1px 半透明 `rgba(255,255,255,0.06)` 线条，避免使用实色分隔
- **动效原则**：所有过渡 200~300ms ease-out，避免突兀跳变

## 三、各区块详细设计提示词

### 3.1 顶部导航栏

**设计提示词**：

> 设计一个深色桌面应用顶部导航栏，高度 56px，背景为半透明深灰 `#12121A` 且带模糊效果。
>
> 左侧：应用 Logo（蝴蝶图标 + "Butterfly Radio" 文字，霓虹青色 #00F0FF）。
>
> 中间偏左：实时日期时间显示，格式为 "HH:MM · YYYY/MM/DD · 星期X"，使用等宽字体，文字颜色为灰紫色 #8888A0。
>
> 右侧区域从左到右依次为：
> 1. **ON AIR 标识**：胶囊型徽章，背景为霓虹红 #FF006E，内有白色文字 "ON AIR" + 脉冲动画圆点，表示正在直播状态
> 2. **LIVE 标识**：胶囊型徽章，背景为霓虹青 #00F0FF，白色文字 "LIVE"
> 3. **QUEUE 按钮**：文字按钮，点击展开播放队列侧栏
> 4. **设置齿轮图标**：使用线条风格图标，悬停时旋转 90 度过渡动画
> 5. **HIDE 按钮**：文字按钮，点击隐藏主内容区域，仅保留迷你播放器
>
> 整体风格：极简、信息密度低、霓虹微光点缀、无多余装饰。

### 3.2 歌曲封面区域

**设计提示词**：

> 设计一个正方形歌曲封面展示区域，尺寸 240×240px，圆角 16px。
>
> 封面图片居中显示，当无封面时显示默认渐变占位图（从 #1A1A28 到 #0A0A0F 的对角渐变，中间放置一个半透明音符图标）。
>
> 封面区域四角有微弱的霓虹青色辉光效果（`box-shadow: 0 0 20px rgba(0,240,255,0.15)`）。
>
> 播放状态下，封面边缘有一圈缓慢旋转的渐变光晕（霓虹青 → 霓虹紫，旋转周期 8 秒）。
>
> 封面右下角叠加一个小的收藏图标（未收藏为线条空心 ♡，已收藏为实心霓虹粉色 ♡）。
>
> 整体风格：暗黑背景上的发光画框效果，赛博朋克感。

### 3.3 频谱可视化区域

**设计提示词**：

> 设计一个音频频谱可视化区域，占据主内容区约 60% 宽度，高度与歌曲封面齐平（约 240px），圆角 16px，背景为 `#0A0A0F`。
>
> 频谱效果：
> - 底部向上生长的柱状频谱条，32~64 根柱体均匀分布
> - 柱体颜色从底部的霓虹青 #00F0FF 渐变到顶部的霓虹紫 #7B61FF
> - 柱体顶部有明亮的高光点
> - 无音频时柱体缓慢起伏（呼吸动画），有音频时随节奏跳动
>
> 背景效果：
> - 深色背景中有微弱的水平扫描线（类似 CRT 效果，透明度 3%）
> - 频谱后方有模糊的光晕扩散效果
>
> 整体风格：赛博朋克音乐可视化器，沉浸感强，视觉冲击力适中（不刺眼）。

## 四、关键交互动效

### 4.1 播放/暂停切换

> 播放按钮按下时：图标从 ▶ 变为 ⏸，按钮背景从 #00F0FF 短暂脉冲放大 1.1x 后回弹（200ms spring 动画）。频谱可视化从静止呼吸模式切换为活跃跳动模式（过渡 500ms）。

### 4.2 歌曲切换

> 切歌时：封面和歌曲信息同时淡出（200ms）→ 新封面和信息淡入（300ms）。进度条归零并从头开始。频谱短暂静默后恢复。气泡区不闪烁。

### 4.3 口播气泡出现

> 新口播出现时：气泡从底部滑入（translateY(10px→0)）+ 淡入（opacity 0→1），持续 400ms ease-out。气泡左侧霓虹青边线有短暂的亮度脉冲。旧气泡向上平移让位。

### 4.4 互动消息发送

> 用户点击发送按钮后：输入框清空 + 按钮纸飞机图标飞出动画（向右上角飞出 + 缩小消失，300ms）。新的用户消息出现在气泡区（右侧对齐，背景为 `rgba(123,97,255,0.15)`，紫色边线）。

### 4.5 音量调节

> 拖动音量滑块时：实时更新音量图标状态（静音🔇→低音量🔈→中音量🔉→高音量🔊）。数字百分比实时跳动。滑块已填充部分宽度实时变化。

### 4.6 收藏操作

> 点击收藏心形图标时：图标从空心线条变为实心，颜色从 #8888A0 变为 #FF006E，同时有心形粒子向四周扩散的小动画（6~8 个小粒子，300ms 消失）。

## 六、响应式适配要点

| 窗口宽度 | 布局调整 |
|---|---|
| ≥ 1440px | 默认布局，封面 240px，频谱区充足 |
| 1280~1439px | 封面缩小至 200px，频谱区等比缩小 |
| 1024~1279px | 封面和频谱改为上下堆叠布局，控制栏紧凑排列 |
| < 1024px | 触发 HIDE 模式自动建议，或切换为单列布局 |

## 七、图标与插画风格指引

- **图标风格**：线条图标（Stroke 1.5px），圆角端点，霓虹色系，参考 Lucide Icons 或 Phosphor Icons 风格
- **空状态插画**：极简线条插画，单色（#3A3A50），赛博朋克元素（电路板线条、波形、音符）
- **Logo 风格**：抽象蝴蝶剪影，翅膀由音频波形线条构成，霓虹青色，可带微弱发光
